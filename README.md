# project
Alarm Clock in Python

This project is a simple Python-based alarm clock that allows the user to:

Select one of multiple alarm sounds

Set an alarm using 24-hour time format (HH:MM:SS)

Play a sound when the alarm time is reached

The project is designed for beginners and uses the built-in winsound module, which works on Windows.

Features

Multiple alarm sound options

Simple console-based interface

Uses accurate system time

Easy to modify and extend

Requirements

This project works on Windows because it uses the winsound module.
How to Use

Download or clone this repository.

Ensure all .wav alarm files are in the same folder as alarm.py.

Run the program:

python alarm.py


Choose an alarm sound when prompted.

Enter the alarm time in the format:

HH:MM:SS


Leave the program running.
The alarm will ring when the time matches.
